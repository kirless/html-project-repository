Introduction

C++ is a programming language of high quality developed at Bell Labs by Bjarne Stroustrup. C++ applies object oriented features to its ancestor, C. C++ is one of the most common programming languages for graphical applications , for instance those running in Windows and Macintosh.

The definition of C ++

It is an object-oriented, structured programming language. It was invented by Byarn Strästrup at Bell Labs. It is a language similar to the C language invented by Dennis Richie in the early 1970s, but it is safer than its predecessor and includes many modern technologies such as Object Oriented programming OOP. 2
